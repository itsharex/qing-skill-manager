2. 扫描流程 — scan_overview 收集全局 + 项目 + 插件 Skill
Composable
useLocalInventory.scanLocalSkills()
→
invoke("scan_overview", { ideDirs, projectDirs })
Rust 后端
scan_overview()
① 仓库扫描
collect_skills_from_dir(manager_dir)
→
manager_skills[] + manager_map[(hash, idx)]
② 版本映射
遍历 manager_skills → get_skill_package()
→
version_hash_map { hash → (versionId, skillId) }
③ IDE 扫描
collect_ide_skills(~/.{ide}/skills/, "global")
→
ide_skills[] (全局)
④ 项目扫描
collect_ide_skills(project/.{ide}/skills/, "project")
→
ide_skills[] (项目级)
⑤ 插件扫描
collect_plugin_skills(~/.claude/plugins/)
→
ide_skills[] (plugin, 只读)
返回
Overview { manager_skills, ide_skills }
3. 版本检测 — collect_ide_skills 内部的同步状态判定
读取 sidecar
read_install_sidecar(skill_dir)
→
{versionId, contentHash} 或 默认空
计算当前 hash
skill_content_hash(SKILL.md)
→
current_hash
判定逻辑
有 sidecar + hash 匹配 → synced ✓
有 sidecar + hash 不同 → modified ⚠
无 sidecar + 仓库匹配 → untracked ?
无匹配 → unknown ✗
输出 IdeSkill
{name, ide, scope, managed, versionId, contentHash, installedHash, syncStatus}
