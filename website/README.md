# Qing Skill Manager Website

Marketing site for **Qing Skill Manager**.

## Purpose

- Present the forked project with its own branding
- Provide a lightweight homepage for GitHub Pages deployment
- Acknowledge the original `skills-manager` project while presenting this fork as an independent package

## Development

```bash
cd website
pnpm install
pnpm dev
```

## Build

```bash
cd website
pnpm build
```

## Deployment Target

- Site: `https://qing-claw.github.io/qing-skill-manager`
- Base path: `/qing-skill-manager`

## Attribution

This site represents a forked and repackaged project built on top of the original
[`skills-manager`](https://github.com/Rito-w/skills-manager).
